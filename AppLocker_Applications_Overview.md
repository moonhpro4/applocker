# AppLocker Applications Overview

## 1. Parent Monitoring Dashboard
- **Real-time Location Tracking**: Features enabling live tracking of the child's location.
- **Wi-Fi Calling**: Functionality for making calls over Wi-Fi for cost-effective communication.
- **Hidden Camera Feed**: A discreet camera feature for monitoring surroundings.
- **Application Monitoring**: Tools to oversee app usage and status on the child’s device.

## 2. Child App Locker
- **Phone/Tablet Lockdown**: Secure lockdown features to restrict access to other applications.
- **Accessibility Service Integration**: Ensures the app can enforce lockdown policies effectively while being compliant with accessibility standards.

## 3. Laptop Child Version
- **Screen Overlay**: Displays information or warnings in a persistent overlay on the screen.
- **App Blocking**: Ability to restrict access to defined applications based on usage or schedule.
- **Folder Locking**: Functionality to lock specific folders to prevent unauthorized access.

## Additional Features
- **Lunara Streaming Integration**: Incorporation of Lunara’s streaming features for media consumption.
- **Secure Encryption**: Ensures that all data handled by the applications is encrypted for security.
